ローカルファイル読み込みの制限について

現在、ブラウザでファイルを直接開いているため（URLが file://...）、セキュリティ上の制限（CORSポリシー）により、各階の JSON データや背景画像を自動で取得できません。

解決方法:

  1. Python等でローカルWebサーバーを起動してアクセスする（推奨）: 
    python -m http.server 8000
    起動後、http://localhost:8000/editor.html にアクセスします。
  2. 右側の「保存・入出力」タブから、編集したい階のJSONファイルを個別に読み込み、背景画像も手動で読み込んで編集します。
背景画像の読み込み

このフロア用の図面背景画像（例: floor2.png）をローカルから読み込んでください。
